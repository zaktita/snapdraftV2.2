# SnapDraft Manual Testing Checklist

## Industry-Specific Test Cases

### 1. E-Commerce / Retail
**Brand Examples:** Fashion boutique, electronics store, handmade crafts
**Test Scenarios:**
- [ ] Upload 5-10 product photos (shoes, clothing, gadgets)
- [ ] Generate "New Arrival" posts for Instagram square
- [ ] Create "Flash Sale" graphics with text overlays
- [ ] Test product badge generation ("20% OFF", "Limited Edition")
- [ ] Generate lifestyle shots from plain product images
- [ ] Create carousel posts (10 slides) for product features
- [ ] Multi-platform campaign (Instagram, Facebook, LinkedIn) for same product

**Expected Quality:**
- Clean, professional product presentation
- Brand colors maintained across all outputs
- Text readability on product images
- Consistent style across carousel slides

## Canvas Editor Testing (All Industries)

### EDT-001: Erase/Remove Elements
- [ ] **E-commerce:** Remove price tag from product photo
- [ ] **Food:** Remove unwanted table items from dish photo
- [ ] **Real Estate:** Remove "For Sale" sign from property
- [ ] **Fitness:** Remove equipment from background
- [ ] **Beauty:** Remove competitor branding from shelf

### EDT-002: Inpaint/Replace Elements
- [ ] **E-commerce:** Replace background color of product shot
- [ ] **Food:** Replace plain plate with decorative one
- [ ] **Real Estate:** Replace furniture in staged room
- [ ] **Fitness:** Replace gym equipment in background
- [ ] **Beauty:** Replace backdrop in treatment photo

### EDT-003: Add Text Overlays
- [ ] **E-commerce:** Add "SALE 50% OFF" to product image
- [ ] **Food:** Add "Chef's Special" to dish photo
- [ ] **Real Estate:** Add price and address overlay
- [ ] **Fitness:** Add "New Class Alert" text
- [ ] **Beauty:** Add "Limited Time Offer" banner

### EDT-004: Adjust Brightness/Contrast
- [ ] **E-commerce:** Brighten underexposed product photo
- [ ] **Food:** Enhance contrast on food photography
- [ ] **Real Estate:** Brighten dark interior shots
- [ ] **Fitness:** Increase contrast on action shots
- [ ] **Beauty:** Soften harsh lighting on face

### EDT-005: Resize/Crop
- [ ] **E-commerce:** Crop product image to focus on details
- [ ] **Food:** Resize landscape dish photo to square
- [ ] **Real Estate:** Crop wide property shot to feature entrance
- [ ] **Fitness:** Resize portrait gym shot to landscape
- [ ] **Beauty:** Crop product photo to zoom on packaging

### EDT-006: Undo/Redo
- [ ] Test undo after each editing action (all industries)
- [ ] Test redo after accidental undo
- [ ] Test multiple undo/redo sequences
- [ ] Verify undo/redo maintains image quality
- [ ] Test undo/redo with combined edits

## Platform-Specific Format Tests

### Instagram
- [ ] **Square (1:1)** - All industries, feed posts
- [ ] **Portrait (4:5)** - All industries, optimized feed posts
- [ ] **Story (9:16)** - All industries, story content

### Facebook
- [ ] **Landscape (1.91:1)** - Professional services, events
- [ ] **Square (1:1)** - Retail, product showcases

### LinkedIn
- [ ] **Landscape (1.91:1)** - B2B, thought leadership
- [ ] **Square (1:1)** - Company updates, job posts

### Twitter/X
- [ ] **Landscape (16:9)** - Tech announcements, news
- [ ] **Square (1:1)** - Quick updates, quotes

## Quality Metrics Checklist

For each industry test, verify:
- [ ] **Brand Color Accuracy:** Original colors maintained ±10%
- [ ] **Style Consistency:** All batch items feel cohesive
- [ ] **Text Readability:** All overlays clear and legible
- [ ] **Image Quality:** No pixelation or artifacts
- [ ] **Composition:** Balanced, professional layouts
- [ ] **Format Compliance:** Correct dimensions for platform
- [ ] **Generation Speed:** <5 seconds per image
- [ ] **Error Handling:** Graceful failures with clear messages

## Recommended Testing Order

### Phase 1: Core Functionality (Day 1)
1. E-commerce batch test (familiar use case)
2. Food & beverage single generation
3. Canvas editor basic edits (erase, text overlay)

### Phase 2: Industry Diversity (Day 2)
4. Real estate multi-platform test
5. Professional services LinkedIn content
6. Fitness weekly calendar
7. Beauty luxury brand test

### Phase 3: Edge Cases (Day 3)
8. Child-friendly content (education)
9. Minimal aesthetic (tech/SaaS)
10. Bold/vibrant (entertainment)
11. Large batch processing (50+ items)

### Phase 4: Platform & Device (Day 4)
12. Mobile testing (iOS + Android)
13. Canvas editor on tablet
14. Different browser testing (Chrome, Safari, Firefox)

### Phase 5: Integration & Performance (Day 5)
15. Payment flow (Paddle sandbox)
16. Email notifications
17. Extended session testing
18. Concurrent user simulation
