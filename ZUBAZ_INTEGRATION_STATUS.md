## 📦 What Changed

### Before (Zubaz Original)

- Dependencies: react-countup, react-player
- Routing: Next.js Link

- Framework: Laravel Inertia + React
- Styling: Tailwind CSS (pure)
- Icons: Lucide React (single library)
- Dependencies: None added (custom implementations)
- Routing: Inertia.js Link
